Настройка Яндекс.Вебмастера

Токен используется для доступа к вашим сайтам в Яндекс.Вебмастере.

Как получить токен?

  1. Перейдите по ссылке инструкции Яндекса.
  2. Зарегистрируйте приложение в Яндекс OAuth.
  3. При регистрации выберите API Яндекс.Вебмастера.
  4. Получите ID приложения и перейдите по ссылке для получения токена.

💡 Альтернатива: Вы можете добавить доступ к вебмастеру для системного логина st.kd@yandex.ru
