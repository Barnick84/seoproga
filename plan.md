Ответ LLM  с идеальной структурой выведи ниже.
Для ручного регулирования структуры, структура должна состоять из секций с заголовком H2, в векцию с заголовком H2, могум входить заголовки h3.

Заголовки можно изменять (переписывать), перемещать, удалять.
Перемещая секцию с заголовком H2,  автоматически перемещаются входящие в секцию заголовки h3.
Заголовок h3  может перемещаться внутри секции, так и в другую секцию H2.
При удалении заголовка h3  из секции, заголовок удаляется.
При удалении заговка h2, если его секция имеет заголовки h3, удалить нельзя, пока из удаляемой секции не переместить все заголовки в другие секции или не удалить.
Пользователь может создавать секции с заголвоками H2, может в секциях с заголовком H2  создавать заголовки h2.
Под каждым заголовком, пользователь может добавть текстовый коментарий, о чем написать в блоке.

Все ключевые слова из кластера пометь, что они учавствовали в оптимизации статьи.
